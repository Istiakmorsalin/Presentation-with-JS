User Interface Design

In our app we followed some of the design patterns of UI for android. Our app uses :

  1. Gesture
  2. Notification Drawer
  3. Action Bar
  4. Navigation
  5. Compound Views and Checklist
  6. Swipe Views
  7. Selection
  8. Widgets
  9. Animation

Class Diagram Design:





We used "Abstract Factory Design Pattern" which Is creational software design pattern. This pattern provides interfaces for creating families of related or dependent objects without specifying their concrete classes

Limitation:

  1. The tips provided by the app is static. We are not able to provide the users any dynamic tips.
  2. The apk size is comparatively larger and that is why is it runs a little bit slow in low memory devices
  3. The Solutions are not obvious

Future Plan

  1. By analyzing sleep cycle, achieving a sleep graph to represent sleeping pattern of a user.
  2. Analyzing the sleep cycle, awake the user in any IREM stage of sleep near awake time.
  3. Sending Dynamic tips to its users.
  4. Developing preloaded voice for the specific callers in database to make them aware that the user is sleeping.
  5. Developing intelligence that can help user scheduling his/her sleep cycle
